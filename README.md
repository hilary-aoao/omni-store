# The Omni Store
A Python-based retail management system built as a personal project


## Features
* **OOP Architecture:** Uses classes and inheritance for Physical and Digital products.
* **Inventory Management:** Tracks stock levels and prevents overselling.
* **Interactive CLI:** A 'While' loop interface for users to view and buy items.
* **List Comprehensions:** Efficiently filters available stock.

## How to Run
1. Clone the repo.
2. Run `python main.py`.
